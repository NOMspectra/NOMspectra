masslib Documentation
=====================

**masslib** is a Python library for working with |ftcir_link| spectra and plotting |vkd_link|.

.. |ftcir_link| raw:: html

    <a href="https://en.wikipedia.org/wiki/Fourier-transform_ion_cyclotron_resonance" target="_blank">FTCIR MS</a>

.. |vkd_link| raw:: html

    <a href="https://en.wikipedia.org/wiki/Van_Krevelen_diagram" target="_blank">Van Krevelen diagrams</a>

Some of the features of masslib are:

* Isotope Distribution generator.
* Assigning brutto formulae to signals.
* Working with spectra as with sets (intersection, union, etc).
